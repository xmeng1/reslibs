# 变更：移除公共上传功能，简化主页UI

## 为什么
将发布功能限制为管理员专用，提升用户体验的简洁性，强化平台作为资源聚合站而非用户生成内容平台的定位。

## 变更内容
- 移除主页中的所有"上传资源"按钮和链接
- 简化主页布局，专注于搜索和资源浏览
- 移除底部链接中的"上传资源"选项
- 保留管理员后台访问入口
- 优化搜索栏位置，使其更加突出

## 影响
- 受影响的规格：ui-design/spec.md
- 受影响的代码：src/app/page.tsx, src/components/navbar.tsx, src/components/footer.tsx
- **BREAKING**: 移除公共用户上传入口，改为仅限管理员操作
