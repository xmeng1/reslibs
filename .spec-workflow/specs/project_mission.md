# Project Mission: ResLibs Unity Platform

## 1. 项目概述
构建一个名为 `unity.reslibs.com` 的资源分享网站，主要聚焦于 Unity Asset 资源的聚合与分享。
核心目标是通过自动化脚本，实现从“原始存储（百度网盘）”到“盈利存储（网赚盘）”的搬运，并利用 AI 自动生成 SEO 友好的内容，最终发布到网站前端。

## 2. 核心价值
* **用户端**：提供一个类似 `koudaizy.com` 的清晰、易检索的资源下载站。
* **管理端**：极度简化的发布流程。管理员只需输入百度网盘链接，系统自动完成下载、转存、内容生成、发布全过程。
* **商业模式**：通过诚通/飞猫/Rapidgator 等网赚网盘的下载收益及网站广告变现。

## 3. 关键功能模块
1.  **Web 前端**：展示资源列表、详情页、搜索、分类。
2.  **Web 后端 (CMS)**：管理资源数据、用户、权限。
3.  **CLI 自动化机器人 (Auto-Bot)**：
    * 百度网盘资源下载。
    * 资源解压与元数据提取（图片、Readme）。
    * LLM 内容增强（基于文件名和原始文件生成介绍）。
    * 多网盘分发上传。
    * 自动发布到 CMS。
