## 🛠️ 技术栈

- **前端框架**: Next.js 14.0.4 (App Router)
- **类型系统**: TypeScript 5
- **样式框架**: Tailwind CSS 3
- **组件库**: Shadcn/UI
- **数据库**: PostgreSQL + Prisma ORM
- **部署**: Vercel (就绪)
- **AI 集成**: Google Gemini API
- **开发工具**: ESLint, Git
